Accept a hierarchical visit of the nodes in the Tiny\+X\+M\+L-\/2 D\+OM. Every node in the \mbox{\hyperlink{classXML}{X\+ML}} tree will be conditionally visited and the host will be called back via the \mbox{\hyperlink{classtinyxml2_1_1XMLVisitor}{X\+M\+L\+Visitor}} interface.
